4. Pada gambar disamping diketahui panjang busur CD = 8 cm, luas juring APB = 24 cm, dan luas juring CPD = 20 cm. Tentukan panjang busur AB Pakai cara ! kalau gak pakai cara saya report :V


Jawaban:

a. Jika panjang busur AB 32 cm, maka panjang busur CD adalah 80 cm.

b. Jika luas juring COD 45 cm², luas Juring AOB adalah 18 cm².

Simak caranya dalam pembahasan.

Pembahasan

Berdasarkan gambar pada soal panjang busur dan luas juring dapat dicari dengan rumus :

Sudut AOB/sudut COD = panjang busur AB/panjang busur CD

Atau

Sudut AOB/sudut COD = luas juring AOB/luas juring COD

Jika panjang busur AB 32 cm, maka panjang busur CD :

Sudut AOB/sudut COD = panjang busur AB/panjang busur CD

48°/120° = 32 cm/panjang busur CD

⅖ = 32 cm/panjang busur CD

panjang busur CD = 32 cm × 5/2 = 80 cm

Jika luas juring COD 45 cm², maka luas Juring AOB :

Sudut AOB/sudut COD = luas juring AOB/luas juring COD

48°/120° = luas juring AOB/45 cm²

⅖ = luas juring AOB/45 cm²

⅖ × 45 cm² = luas juring AOB

18 cm² = luas juring AOB

15. Perhatian gambar diatas Lingkaran di atas memiliki jari-jari sebesar 7 cm. Hitunglah luas juring CPD ?


A.Langkah Pertama 
Pertama-tama kita mencari Luas Juring APB yang dibandingkan Luas Lingkaran beserta sudut-sudutnya :

Luas Juring APB = α°/360° x πr2

Luas Juring APB = 90°/360° x 22/7 . 7^2

Luas Juring APB = 1/4 x 154 = 38,5 cm2 

B.Langkah Kedua 
Pada langkah ke-2 ini kita akan membandingkan kedua luas juring :

Luas Juring APB/Luas Juring CPD = Sudut Juring APB/Sudut Juring CPD

38.5/Luas Juring CPD = 90°/60°

Luas Juring CPD = 38.5 x 2/3

Luas Juring CPD = 25.66 cm2

Jadi Luas Juring CPD adalah 25.66 cm2

Luas juring CPD = α/360 x πr²

                            = 60/360 x 22/7 x 7 x 7

                            = 1/6 x 22 x 7

                            = 154/6

                            = 25,67 cm²
