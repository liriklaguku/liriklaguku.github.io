16. ubah kalimat dari type 1 ke 2 A. if precila composes the sentimental song , she will cry in making the arransement B. if you say yes , i will give it


Task and answer :

Ubah kalimat dari conditional type 1 ke 2!

A. If Precila composes the sentimental song , she will cry in making the arrangement.

Type 2 : If Precila composed the sentimental song , she would cry in making the arrangement.


B. If you say yes, I will give it.

Type 2 : If you said yes, I would give it.

----------

Explanation :

Conditional sentence adalah kalimat pengandaian. Ada tiga tipe kalimat pengandaian :

Type 1 → untuk kejadian yang masih mungkin terjadi di masa yang akan datang (future possible).

Construction : IF + S + infinitive/infinitive+s, S + will + infinitive


Type 2 → pengandaian untuk peristiwa yang terjadi di masa sekarang (present unreal).

Construction : IF + S + preterite (v₂), S + would + infinitive (v₁)


Type 3 → pengandaian untuk peristiwa yang terjadi di masa lampau (past unreal).

Construction : IF + S + had + past participle (v₃), S + would + have + past participle (v₃)

-----------

Answer Details

Subject : English

Category : Structure and written expressions

Sub-category : Conditional clause

Level : JHS VIII

Code : 8.5

Keywords : if, type 1, type 2, conditional

---------

ubah kalimat dari type 1 ke 2
A. if Priscilla composes the sentimental song , she will cry in making the arrangement
= if Priscilla composed the sentimental song, she would cry in making the arrangement

B. if you say yes, I will give it
= If you said yes, I would give it

Rumus Conditional Type 2
If + Subject + Verb 2 + Subject + would/could/might + Verb 1
