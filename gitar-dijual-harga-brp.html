Struktur Kalimat

Kalimat Pembuka :

Pembeli : "Bu saya mau beli gitar ini, berapa harganya?"

Penjual : "Kalau gitar yang itu harganya Rp750.000,00 Nak.

Isi :

Pembeli: "Harganya boleh kurang tidak, Bu?"

Penjual : "Hmmm, boleh. Mau nawar berapa, Nak?"

Pembeli: "Rp600.000,00 saja Bu, bagaimana?"

Penjual : "Wah, harga itu rasanya tidak bisa, Nak."

Pembeli: "Kalau Rp625.000,00?" Penjual: "Naikkan sedikit Nak, Rp650.000,00, Ibu lepas gitar ini."

Penutup

Pembeli : "Iya deh Bu, saya setuju, ini uangnya."

Penjelasan:

Struktur Teks Negosiasi :

1. Orientasi :

Pembeli : "Bu saya mau beli gitar ini, berapa harganya?"

Penjual : "Kalau gitar yang itu harganya Rp750.000,00 Nak.

Pembeli : "Harganya boleh kurang tidak, Bu?"

2. Pengajuan :

Penjual : "Hmmm, boleh. Mau nawar berapa, Nak?"

3. Penawaran :

Pembeli: "Rp600.000,00 saja Bu, bagaimana?"

Penjual : "Wah, harga itu rasanya tidak bisa, Nak."

Pembeli: "Kalau Rp625.000,00?"

Penjual: "Naikkan sedikit Nak, Rp650.000,00, Ibu lepas gitar ini."

4. Persetujuan :

Pembeli : "Iya deh Bu, saya setuju, ini uangnya."
